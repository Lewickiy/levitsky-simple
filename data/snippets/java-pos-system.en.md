# POS system для сети кофеен

Это мой первый коммерчески полезный проект, написанный для владельца сети кофеен в Санкт-Петербурге.

## Описание

Проект представляет собой desktop POS-систему для бариста:
- учёт товарных остатков
- продажи
- работа без интернета

## Технологии

- Java
- JavaFX
- Hibernate
- MySQL
- SQLite

## Offline-режим

SQLite использовался для хранения продаж при отсутствии соединения.
